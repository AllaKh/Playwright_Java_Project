Suite‑room code mismatch between Booking flow and Admin → Report

Summary:
When a guest books the Suite, the confirmation page shows room code 104, yet the same reservation appears under room code 103 on the Admin → Report screen. The discrepancy leads to reporting errors and possible overbooking.

Steps to Reproduce:

  1. Navigate to /reservation/3 (Suite) with any valid future check‑in and check‑out dates.
  2. Complete the two‑step booking form.
      ↳ Observed: Confirmation banner states “Room: 104”.
  3. Log in as admin (admin / password) and open Admin → Report.

Actual Result:
The new entry is listed as “Room: 103”.

Expected Result:
The room code in the report must match the code shown to the guest (104) and the physical allocation in the inventory system.

Impact / Severity: Critical
Misreporting room numbers can cause double booking, housekeeping confusion, and revenue‑management inaccuracies. The mismatch must be resolved before the Suite can be offered reliably.
